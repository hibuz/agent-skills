agent-skills:code-simplification skill을 invoke하세요.

정확한 동작을 보존하면서 최근 변경된 코드(또는 지정된 범위)를 단순화하세요:

1. CLAUDE.md를 읽고 프로젝트 규약을 학습
2. 대상 코드 식별 — 더 넓은 범위가 지정되지 않는 한 최근 변경
3. 손대기 전에 코드의 목적, 호출자, 엣지 케이스, 테스트 커버리지를 이해
4. 단순화 기회 스캔:
   - 깊은 중첩 → guard clause 또는 추출된 헬퍼
   - 긴 함수 → 책임별로 분할
   - 중첩 ternary → if/else 또는 switch
   - 일반적인 이름 → 서술적인 이름
   - 중복 로직 → 공유 함수
   - 죽은 코드 → 확인 후 제거
5. 각 단순화를 점진적으로 적용 — 변경마다 테스트 실행
6. 모든 테스트가 통과하고, build가 성공하고, diff가 깨끗한지 검증

단순화 후 테스트가 실패하면, 그 변경을 되돌리고 재고하세요. 결과 리뷰에는 `code-review-and-quality`를 사용하세요.
